# Desafio Filmes

## Contexto
Crie uma listagem de filmes seguindo o mesmo padrao usado em `app/produtos`.

## O que montar
- Uma pagina com uma lista de filmes.
- Um link em cada filme para uma rota dinamica com `id`.
- O envio de `nomeFilme` e `genero` pela URL.
- Uma pagina de detalhe exibindo essas informacoes.

## Dados que devem passar na URL
- `nomeFilme`
- `genero`

## Resultado esperado
Ao clicar em um filme, o usuario deve abrir a pagina de detalhe e visualizar o titulo e o genero.
