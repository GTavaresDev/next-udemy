# Desafio Zoologico

## Contexto
Crie uma listagem de animais de um zoologico no mesmo formato usado em `app/produtos`.

## O que montar
- Uma pagina com uma lista de animais.
- Um link em cada animal para uma rota dinamica com `id`.
- O envio de `animal` e `nivelPerigo` pela URL.
- Uma pagina de detalhe exibindo essas informacoes.

## Dados que devem passar na URL
- `animal`
- `nivelPerigo`

## Resultado esperado
Ao clicar em um animal, o usuario deve abrir a pagina de detalhe e visualizar o nome do animal e o nivel de perigo.
