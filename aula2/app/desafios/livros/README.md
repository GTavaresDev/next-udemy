# Desafio Livros

## Contexto
Crie uma listagem de livros no mesmo formato usado em `app/produtos`.

## O que montar
- Uma pagina com uma lista de livros.
- Um link em cada livro para uma rota dinamica com `id`.
- O envio de `nomeLivro` e `autor` pela URL.
- Uma pagina de detalhe exibindo essas informacoes.

## Dados que devem passar na URL
- `nomeLivro`
- `autor`

## Resultado esperado
Ao clicar em um livro, o usuario deve abrir a pagina de detalhe e visualizar o nome do livro e o autor.
