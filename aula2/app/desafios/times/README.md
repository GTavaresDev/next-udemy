# Desafio Times

## Contexto
Crie uma listagem de times no mesmo formato usado em `app/produtos`.

## O que montar
- Uma pagina com uma lista de times.
- Um link em cada time para uma rota dinamica com `id`.
- O envio de `nomeTime` e `cidade` pela URL.
- Uma pagina de detalhe exibindo essas informacoes.

## Dados que devem passar na URL
- `nomeTime`
- `cidade`

## Resultado esperado
Ao clicar em um time, o usuario deve abrir a pagina de detalhe e visualizar o nome do time e a cidade.
