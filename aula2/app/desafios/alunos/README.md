# Desafio Alunos

## Contexto
Crie uma listagem de alunos no mesmo formato usado em `app/produtos`.

## O que montar
- Uma pagina com uma lista de alunos.
- Um link em cada aluno para uma rota dinamica com `id`.
- O envio de `nomeAluno` e `nota` pela URL.
- Uma pagina de detalhe exibindo essas informacoes.

## Dados que devem passar na URL
- `nomeAluno`
- `nota`

## Resultado esperado
Ao clicar em um aluno, o usuario deve abrir a pagina de detalhe e visualizar o nome e a nota.
